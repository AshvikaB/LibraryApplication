Download the minilib_db dump file and minilib.py file
Open the sql scripts on MySQLWorkbench
Run the scripts using the lightning bolt icon. The database and all the tables should be seen on the left under SCHEMAS. All procedures should also be present under the procedures section
Go to terminal and install PyMySQL using the following command if not already installed: pip install pymysql
Open Anaconda, PyCharm, or another text editor that supports Python to run the minilib_app.py file
Pandas and matplotlib should be installed in your python application. Make sure that the pandas and matplotlib libraries are installed. The commands are: pip install pandas and pip install matplotlib
Run the file
Enter inputs for the prompts generated from the python file. Refer to the user flow for specific details
